# The Colombian Way

## Description
The Colombian Way is a travel website designed to help users explore and plan trips to Colombia. The website provides information about popular destinations and travel packages tailored for families, couples, and solo travelers.

## Features
- Information about Colombia's top destinations:
  - Bogotá
  - Medellín
  - Cartagena
- Travel package options:
  - Family Package
  - Couples Package
  - Solo Traveler Package
- Responsive and user-friendly design
- Navigation menu for easy browsing

## Technologies Used
- HTML5
- CSS3
- GitHub Pages

## Accessibility Features
- Semantic HTML elements
- Responsive layout
- Clear navigation structure
- Accessible text and color contrast
